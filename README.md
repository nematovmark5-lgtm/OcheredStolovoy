# Система управления столовой

**В РАЗРАБОТКЕ ПРОЕКТА УЧАСТВОВАЛ:** Нематов Марк

## Описание

Система управления столовой — это инновационное веб-приложение для создания и управления заказами еды с современным подходом, созданное с использованием Flask и передовых веб-технологий. Приложение предоставляет полнофункциональную платформу для автоматизации процессов столовой с разделением на три специализированных модуля.

## Основные возможности

### Пользовательский интерфейс
- Современный дизайн — стильный UI с анимированными переходами
- Адаптивная верстка — оптимизация для всех типов устройств
- Интуитивная навигация — простой и понятный интерфейс
- Визуальные карточки — красивое отображение блюд и заказов
- Цветовая схема — профессиональная палитра с акцентами

### Модуль оформления заказов
- Создание новых заказов с выбором блюд и напитков
- Валидация выбора — ограничение по одному блюду и одному напитку
- Анимированная корзина — визуальные эффекты добавления товаров
- Система автоматической нумерации — уникальные номера заказов
- Процесс оплаты — имитация банковской транзакции

### Монитор кухни
- Отслеживание заказов в реальном времени
- Таймеры приготовления с визуальными индикаторами
- Управление статусами заказов
- Система сортировки — по номеру или времени заказа
- Прогресс-бары приготовления

### Окно выдачи заказов
- Раздельное отображение заказов по статусам
- Звуковые оповещения о готовых заказах
- Автоматическое обновление каждые 2 секунды
- Визуальное выделение новых заказов

## Технологический стек

### Backend
- Python 3.8+ — основной язык программирования
- Flask 2.3+ — веб-фреймворк
- MySQL Connector — драйвер для работы с базой данных
- Flask-CORS — обработка CORS запросов

### Frontend
- HTML5 — семантическая разметка страниц
- CSS3 — современные стили и адаптивный дизайн
- JavaScript (ES6+) — асинхронные запросы и DOM-манипуляции
- Font Awesome — векторные иконки

### База данных
- MySQL 8.0+ с поддержкой UTF-8
- Таблица orders со структурой:
  - id INT AUTO_INCREMENT PRIMARY KEY
  - number_order INT NOT NULL UNIQUE — номер заказа
  - food VARCHAR(255) NOT NULL — выбранное блюдо
  - drink VARCHAR(255) NOT NULL — выбранный напиток
  - status ENUM — статус заказа
  - order_time TIMESTAMP — время создания заказа

## Функции системы

### Управление заказами
- Создание новых заказов через веб-интерфейс
- Автоматическая генерация номеров заказов
- Отслеживание статусов в реальном времени
- Обновление статусов сотрудниками кухни
- Завершение и архивирование выполненных заказов

### Система статусов
- Готовится — заказ принят и находится в процессе приготовления
- Готово — заказ приготовлен и ожидает выдачи
- Отдал заказ — заказ выдан клиенту

### Интеграция и синхронизация
- RESTful API для взаимодействия между модулями
- Автоматическое обновление данных каждые 2 секунды
- Синхронизация статусов между всеми клиентами
- Обработка конкурентных запросов

## API Endpoints

### Основные endpoints
- GET /api/last_order — получение следующего номера заказа
- POST /api/order — создание нового заказа
- GET /api/orders — получение всех заказов
- GET /api/kitchen_orders — получение заказов для монитора кухни
- POST /api/mark_ready — отметка заказа как готового
- POST /api/complete_order — завершение заказа

Установка и запуск
Предварительные требования
Python 3.8 или выше

MySQL Server 8.0 или выше

Установленный pip

Шаги установки
Установите необходимые зависимости:

bash
pip install flask mysql-connector-python flask-cors
Настройте базу данных MySQL:

sql
CREATE DATABASE stolovaya;
CREATE USER 'Mark'@'localhost' IDENTIFIED BY '0987654321';
GRANT ALL PRIVILEGES ON stolovaya.* TO 'Mark'@'localhost';
Создайте таблицу для заказов:

sql
USE stolovaya;

CREATE TABLE orders (
    id INT AUTO_INCREMENT PRIMARY KEY,
    number_order INT NOT NULL UNIQUE,
    food VARCHAR(255) NOT NULL,
    drink VARCHAR(255) NOT NULL,
    status ENUM('Готовится', 'Готово', 'Отдал заказ') DEFAULT 'Готовится',
    order_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
Запустите сервер:

bash
python СЕРВЕР.py
Откройте в браузере:

Интерфейс оформления заказов: Оформления_Талона.html

Монитор кухни: Кухня.html

Окно выдачи заказов: Окно_заказов.html
